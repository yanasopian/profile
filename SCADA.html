What is Scada ?

Supervisory control and data acquisition (SCADA) is a control system architecture that includes various software and hardware elements such as computers, data communications, peripheral devices, and controllers that interface with machinery.(www.realpars.com)

What is HMI and SCADA ?

SCADA acts as a central database that stores and manages operations on a large scale. It works in tandem with the PLC, which facilitates communication and operation within the system. HMI is an interface that allows a human operator to interact with the whole SCADA system using screens, buttons, levers, or keyboards.(www.realpars.com)
